## 方法1
在 app.js 中加上
`global.name = '张三'`
在其他组件中调用
`global.name`
## 方法2
在 app.js 中加上
`wx.name = '张三'`
在其他组件中调用
`wx.name`
## 方法3
在 app.js 中加上
```js
App({
	globalData: {
	  userInfo: null,
	  userName: '张三'
	}
})
```
在其他组件中调用
```js
const app = getApp()
app.globalData.userName
```
## 总结
方法 1、2 不用每次都获取 app 实例，比较方便，但是因为是全局变量所以容易污染顶级变量。第三种需要每次都获取 app 实例，但是不污染顶级变量，大家视情况使用。
