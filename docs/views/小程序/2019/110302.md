## 需求
今天做小程序的时候遇到一个问题，那就是图片走的 cdn，但是图片又是放在项目中的 image 文件夹里，这样的话打包后图片也会被打包，结果就会提示打包后文件超过 2M，所以我的需求是打包时不上传 image 文件夹。
## 解决
找了好久，才发现官方文档中有这个选项。
在项目的 project.config.json 文件中加上 packOptions 选项：
```json
{
	"description": "项目配置文件",
	"packOptions": {
		"ignore": [
	      {
	        "type": "folder",
	        "value": "images"
	      }
        ]
    ]
}
```
