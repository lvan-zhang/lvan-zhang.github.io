## 卸载gitlab
- 1.停止gitlab
`gitlab-ctl stop`
- 2.卸载gitlab
`rpm -e gitlab-ee`
- 3.查看gitlab进程
`ps -ef|grep gitlab`
杀掉第一个守护进程
`kill -9 16680`
- 4.删除gitlab文件
`find / -name gitlab|xargs rm -rf `
删除所有包含gitlab的文件及目录，删除gitlab-ctl uninstall时自动在root下备份的配置文件
`rm -f /root/gitlab*`
