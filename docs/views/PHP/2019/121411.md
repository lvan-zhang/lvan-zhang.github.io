## 什么是扩展
php扩展就是php核心并不支持的功能，然后可以通过扩展的方式进行扩展PHP的功能，常见的扩展如MySQL，gb2等等。
## 查看php安装了那些扩展
方法一：通过phpinfo()函数，进行查看
方法二：执行php -m 命令：`php -m`
