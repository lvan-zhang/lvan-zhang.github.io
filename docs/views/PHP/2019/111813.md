- php 的 api 都是全局变量，不像 js 中 `String.xxx` ，这也是常常被人诟病的一件事
- php5.4 之后的数组也可以像 js 这样定义：`$arr = []`
- `php -a` 可以进入php的命令行交互
- `??` 和 `?:`
	```php
	$a = 0;
	
	$a ?? 2; 
	相当于 isset($a) ? $a : 2; //结果是0
	$a ?: 2; 
	相当于 !empty($a) ? $a : 2; //结果是2
	```
